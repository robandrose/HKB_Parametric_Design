## Einführung Processing ### Geschichte: Processing wurde von Ben Fry und Casey Reas 2001 am MIT Media Lab entwickelt, quasi als direkter Nachfolger des «Disign by Numbers» Projekt von John Maeda, in deren Klasse sie studierten. Das Ziel war es eine vereinfachte Programmierumgebung zu entwickeln um einen einfachen Einstieg in die Programmierung zu ermöglichen.
### Basis Die Basis von Processing ist die Programmiersprache JAVA. Processing ist eine Bibliothek dieser Sprache und hat alle Möglichkeiten, die auch Java bietet. Java wird in einer sogenannten Runtime-Engine ausgeführt, dadurch können die selben Programmierbefehle auf allen Betriebssystemen ausgeführt werden, für welche eine solche Runtime Engine existiert (Mac, Windows, Linux, Android).
